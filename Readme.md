# FLORESCER
### Conectando mulheres, fortalecendo redes

Plataforma digital para centralização e visibilidade de coletivos femininos de São João da Boa Vista e região.

---

## Sobre o projeto

O FLORESCER é uma plataforma web responsiva desenvolvida como projeto de extensão universitária do curso de **Tecnologia em Sistemas para Internet** do **IFSP — Campus São João da Boa Vista**, no âmbito da Curricularização da Extensão 2026/2027.

O projeto nasce de uma constatação simples: os coletivos femininos da região existem, atuam e transformam vidas — mas são invisíveis digitalmente. Mulheres em situação de vulnerabilidade não conseguem encontrá-los. As lideranças não têm tempo nem suporte técnico para manter presença digital de forma descentralizada.

O FLORESCER resolve isso com um hub digital que centraliza, organiza e conecta.

---

## Funcionalidades

- Diretório de coletivos femininos com página de perfil individual
- Mapa interativo com localização dos coletivos e serviços de apoio
- Calendário de eventos e reuniões
- Lista de contatos de emergência e apoio
- Busca por nome ou área de atuação
- Painel administrativo para atualização autônoma pelas lideranças
- Autenticação segura via e-mail

## Acessibilidade

O projeto segue as diretrizes **WCAG 2.1** com foco em:

- Contraste mínimo de 4.5:1 para textos
- Navegação por teclado
- Compatibilidade com leitores de tela
- Tamanho mínimo de fonte 16px
